# 🎨 Apresentação de Desenhos - Cartoon Network

## 📖 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web utilizando HTML5 e CSS3.

O site apresenta informações sobre alguns desenhos clássicos da Cartoon Network, permitindo que o usuário conheça detalhes como criador, ano de lançamento, gênero e sinopse de cada obra. Além disso, cada página disponibiliza um link para uma plataforma de streaming onde o desenho pode ser assistido.

---

## 🚀 Funcionalidades

- Página inicial com navegação para os desenhos selecionados.
- Página individual para cada desenho.
- Exibição de:
  - Criador
  - Ano de estreia
  - Gênero
  - Sinopse
- Links externos para plataformas de streaming.
- Página "Sobre" apresentando o autor e o projeto.
- Interface com imagens de fundo personalizadas para cada desenho.

---

## 📺 Desenhos Apresentados

- Hora de Aventura
- Apenas um Show
- O Incrível Mundo de Gumball
